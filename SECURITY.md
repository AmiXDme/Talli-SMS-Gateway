# Security Policy

## Supported Versions

Only the latest version of Talli is currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security seriously at Talli Labs. If you discover a security vulnerability within this project, please let us know immediately.

To report a vulnerability, please follow these steps:

1.  **Do not open a public issue.** Security vulnerabilities should be reported privately to ensure the safety of our users.
2.  **Email the project maintainer** (Check Git config or Profile) with a detailed description of the vulnerability.
3.  Include steps to reproduce the issue, potential impact, and any suggested fixes.

We will acknowledge receipt of your vulnerability report within 48 hours and provide a timeline for a fix. Thank you for helping keep Talli secure for everyone!

## Our Commitment

*   We will investigate all reports and provide a timely response.
*   We will keep the reporter informed of our progress.
*   We will give credit to the reporter if desired once the vulnerability is patched.
